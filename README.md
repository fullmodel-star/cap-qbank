# 📝 會考英語題庫（CAP）

一個**離線可用、免安裝、零追蹤**的國中教育會考英語**歷屆題庫**練習 PWA，收錄官方題目，支援整卷模擬、題型練習、弱點分析與錯題本。含**聽力音檔**。

> 純單一 HTML 檔（題目內嵌）、資料全存本機、免註冊、無廣告。

> **狀態**：✅ 已上線　｜　🟣 靛藍主題（theme `#4338ca`）　｜　官方 **110–115** 題庫、**226 題閱讀 + 6 年聽力**　｜　📱 響應式
> 上線網址：<https://fullmodel-star.github.io/cap-qbank/>　｜　已收錄於[英語學習入口 hub](../14_英語學習入口/)
> 原始題庫來源：`E:\英語資料庫`

---

## ✨ 功能
- **整卷模擬**：依年份完整作答、計分
- **題型練習**：分題型針對性練習
- **弱點分析 / 錯題本**：找出弱點、收錄答錯題重練
- **聽力**：內建 6 年聽力音檔（`audio/`）

## 🚀 使用
瀏覽器開 `index.html` 即可；安裝成 App 需用上線 https 網址（Android Chrome →「安裝應用程式」，iPhone Safari →「加入主畫面」）。

## 📁 檔案結構
| 檔案 | 說明 |
|---|---|
| `index.html` | 主程式（題目內嵌，單檔運作） |
| `audio/` | 聽力音檔 |
| `manifest.json` / `sw.js` | PWA 設定與離線快取 |
| `icon-*.png` / `apple-touch-icon.png` / `icon.svg` | 圖示 |

> 原始 PDF 試題與題庫 JSON 保存於 `E:\英語資料庫`（開發／重建用）。

## 🔒 隱私
完全離線、不連外部伺服器；作答紀錄只存裝置 `localStorage`；無帳號、無追蹤、無廣告。
